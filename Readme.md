# Daily Comic Practice Cycle

Simple static website for the 10 day drawing and comic practice cycle from
`Daily Structure.pdf`.

## How to Use

- Open `index.html` in a browser.
- The cycle starts on June 23, 2026 as day 1 and repeats every 10 days.
- Change the check date to see which cycle day any date belongs to.
- Use the day buttons to preview any topic in the cycle.

## Features

- 10 day rotating topic cycle.
- Daily practice structure for warm-up, skill study, application, and project work.
- Per-day topic details based on the PDF.
- Short, mid-range, and long-range project notes.
- Fully static: no build step and no internet connection required.
